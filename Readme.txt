How to use
---------------------------------------------------------------------------------------------------------------------------------------
After you start the application, left-click on any grid to make it alive. After arranging the cells as needed, press spacebar and this will start the simulation.
On simulation finish, you can press ESC and re-enter it again and start another simulation as normal.
